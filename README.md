# Project Overview

Zagi refers to the Zagi Retail Company case study used in database courses to demonstrate relational design, SQL queries, and data warehousing.

## Repository Contents
ayelezm_zagimore.sql: The primary transactional database schema and data (OLTP).

ayelezm_zagimore_ds.sql: This is "Data Staging" where the scripts are used to prepare data for transformation.

ayelezm_zagimore_dw.sql: The Data Warehouse schema (OLAP), features a Star Schema with dimension and fact tables.
